# Командная строка Ubuntu/Linux

## Файлы и папки

- `ls` - список файлов и папок в текущей директории
- `ls -la` - подробный список с скрытыми файлами
- `cd <путь>` - перейти в указанную папку
- `cd ..` - перейти в родительскую папку
- `cd ~` - перейти в домашнюю директорию
- `cd /` - перейти в корневую директорию
- `mkdir <имя>` - создать новую папку
- `rm <файл>` - удалить файл
- `rm -r <папка>` - удалить папку рекурсивно
- `rm -rf <папка>` - принудительное удаление без подтверждения
- `cp <исходный> <целевой>` - копировать файл/папку
- `cp -r <исходный> <целевой>` - копировать папку рекурсивно
- `mv <исходный> <целевой>` - переместить или переименовать файл/папку

## Права доступа

- `chmod +x <файл>` - сделать файл исполняемым
- `chmod 755 <файл>` - установить права (владелец: rwx, группа: rx, другие: rx)
- `chmod 644 <файл>` - установить права (владелец: rw, группа: r, другие: r)
- `chown <пользователь>:<группа> <файл>` - изменить владельца файла
- `sudo <команда>` - выполнить команду с правами суперпользователя

## Работа с текстом

- `cat <файл>` - вывести содержимое текстового файла
- `less <файл>` - просмотр файла с прокруткой
- `head <файл>` - вывести первые 10 строк файла
- `tail <файл>` - вывести последние 10 строк файла
- `tail -f <файл>` - отслеживать изменения файла в реальном времени
- `grep "текст" <файл>` - поиск текста в файле
- `echo "текст" > файл.txt` - создать файл с текстом (перезаписать)
- `echo "текст" >> файл.txt` - добавить текст в конец файла
- `nano <файл>` - редактировать файл в простом текстовом редакторе
- `vim <файл>` - редактировать файл в продвинутом редакторе

## Дерево папок
```bash
sudo apt install tree
```
- `tree` - показать дерево папок и файлов
- `tree -d` - показать только дерево папок (без файлов)
- `tree -L 2` - показать дерево глубиной 2 уровня
- `find . -type f -name "*.txt"` - найти все txt файлы в текущей директории
- `find /home -name "файл"` - найти файл по имени в указанной директории

## Система и процессы

- `ps aux` - список всех запущенных процессов
- `top` - интерактивный просмотр процессов (аналог диспетчера задач)
- `htop` - улучшенная версия top (требует установки)
- `kill <PID>` - завершить процесс по ID
- `kill -9 <PID>` - принудительно завершить процесс
- `pkill <имя_процесса>` - завершить процесс по имени
- `df -h` - информация о свободном месте на дисках
- `du -sh <папка>` - размер папки
- `free -h` - информация об использовании оперативной памяти
- `uname -a` - информация о системе
- `neofetch` - красивый вывод информации о системе (требует установки)

## Сеть

- `ifconfig` или `ip addr` - информация о сетевых интерфейсах
- `ping <адрес>` - проверить соединение с узлом
- `traceroute <адрес>` - трассировка маршрута
- `netstat -tulpn` - список активных сетевых подключений и портов
- `ss -tulpn` - современная альтернатива netstat
- `wget <url>` - скачать файл из интернета
- `curl <url>` - отправить HTTP запрос или скачать файл

## Пакеты и установка программ

- `sudo apt update` - обновить список пакетов
- `sudo apt upgrade` - обновить установленные пакеты
- `sudo apt install <пакет>` - установить пакет
- `sudo apt remove <пакет>` - удалить пакет
- `dpkg -l` - список установленных пакетов
- `snap find <приложение>` - поиск приложений в Snap Store

## Пользователи и права

- `whoami` - текущий пользователь
- `id` - информация о текущем пользователе и группах
- `groups` - группы текущего пользователя
- `sudo su` - переключиться на root
- `su <пользователь>` - переключиться на другого пользователя
- `passwd` - изменить пароль текущего пользователя
- `sudo passwd <пользователь>` - изменить пароль другого пользователя

## Архивация

- `tar -czf архив.tar.gz папка/` - создать tar.gz архив
- `tar -xzf архив.tar.gz` - распаковать tar.gz архив
- `zip архив.zip файлы` - создать zip архив
- `unzip архив.zip` - распаковать zip архив

## Дополнительно

- `man <команда>` - справка по команде (manual)
- `<команда> --help` - краткая справка по команде
- `clear` или `Ctrl+L` - очистить экран терминала
- `history` - история команд
- `!!` - повторить последнюю команду
- `Ctrl+C` - прервать выполнение команды
- `Ctrl+Z` - приостановить выполнение команды
- `jobs` - список приостановленных задач
- `fg` - вернуть приостановленную задачу на передний план
- `exit` - закрыть терминал

## Полезные комбинации

- `ls -la | grep "паттерн"` - поиск файлов по паттерну
- `cat файл | grep "текст"` - поиск текста в файле
- `command > файл.txt` - перенаправить вывод в файл (перезаписать)
- `command >> файл.txt` - перенаправить вывод в файл (дописать)
- `command1 | command2` - передать вывод первой команды на вход второй
